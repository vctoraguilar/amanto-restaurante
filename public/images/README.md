## Especificaciones técnicas recomendadas:

- **Formato**: JPG para fotos, PNG para logos con transparencia
- **Resolución**: Mínimo 1920x1080px para imágenes principales
- **Optimización**: Comprimir imágenes para web (mantener calidad pero reducir tamaño)
- **Alt text**: Describir cada imagen para accesibilidad

## Notas:

- Todas las imágenes deben ser de alta calidad y profesional
- Mantener consistencia en el estilo fotográfico
- Usar iluminación natural cuando sea posible
- Asegurar que las imágenes representen fielmente los platos y el ambiente del restaurante
